题目背景：
工厂，或者类似的企业中，每天会有产品的进来与出口，需要用一个软件，去记录产品的进厂出厂的数据，并且进行数据的存储和打印录入

实现意义：
能够让工厂的产品信息更加的清楚完整，和保存

主要问题：

多客户端输入，会存在数据库并发访问的情况，需要对数据库加锁

数据的备份和恢复
